漢新二手車專賣網站
這是一個二手車買賣公司的官方網站，提供車輛展示、汽車知識、預約賞車等功能。

專案概述
漢新二手車專賣網站是一個靜態網站，展示公司服務、車輛庫存、汽車相關知識，並提供預約賞車功能。網站設計以用戶體驗為中心，提供清晰的分類和直觀的導航。

首頁 (index.html)：網站入口，展示最新車款、公司介紹、服務項目

知識庫 (knowledge.html)：汽車保養、安全駕駛相關知識（待建立）

聯絡我們 (contact.html)：聯絡資訊與預約表單（待建立）

安裝與使用
本地運行
將所有檔案下載到本地目錄

確保圖片檔案位於 images/ 目錄下

使用瀏覽器直接打開 index.html 即可瀏覽網站

注意事項
部分連結頁面尚未建立（標記為「待建立」）

預約表單僅為前端演示，無後端處理功能

所有車輛價格與資訊僅為示例

網站頂部有「這是網頁測試，尚未開始營業」的通知
